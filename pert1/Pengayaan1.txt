Pertanyaan Refleksi
Setelah menyelesaikan kode, jawab pertanyaan berikut:

Jelaskan perbedaan antara fungsi (function) dan prosedur (procedure) dalam pemrograman berorientasi objek.
Mengapa penting menggunakan konstanta (final variables) dalam suatu kelas? Apa keuntungannya?
Apa yang terjadi jika kita mencoba mengubah nilai suatu konstanta setelah objek dibuat? Berikan contoh.
Bagaimana cara mendefinisikan konstanta dalam bahasa Java?

Jawaban

1. Perbedaan antara Fungsi dan Prosedur:
   - FUNGSI (Function): Method yang mengembalikan nilai (return value). Ditandai dengan tipe return yang bukan void.
     Contoh: double hitungBMI() - mengembalikan nilai BMI bertipe double
             double hitungKalori() - mengembalikan nilai kalori bertipe double
   
   - PROSEDUR (Procedure): Method yang tidak mengembalikan nilai. Ditandai dengan tipe return void.
     Contoh: void tampilKategori() - hanya menampilkan kategori tanpa mengembalikan nilai
             void tampilInfo() - hanya menampilkan informasi tanpa mengembalikan nilai

2. Pentingnya Konstanta (final variables):
   - KEAMANAN DATA: Melindungi nilai-nilai penting agar tidak berubah secara tidak sengaja
   - STANDARISASI: Memastikan nilai tertentu (seperti faktorAktivitas = 1.2) konsisten di seluruh program
   - DOKUMENTASI: Memberikan makna pada nilai-nilai tetap (lebih jelas daripada hardcode)
   - PEMELIHARAAN: Jika perlu mengubah nilai konstanta, cukup ubah di satu tempat (deklarasi)
   - PERFORMA: Compiler dapat mengoptimalkan kode karena tahu nilai tidak akan berubah

3. Mengubah Nilai Konstanta:
   Jika kita mencoba mengubah nilai konstanta setelah objek dibuat, akan terjadi COMPILE ERROR.
   
   Contoh:
   Pengguna p = new Pengguna();
   p.faktorAktivitas = 1.5;  // ERROR!
   
   Pesan error dari compiler:
   "cannot assign a value to final variable faktorAktivitas"
   
   Ini mencegah kesalahan programming dan memastikan integritas data.

4. Cara Mendefinisikan Konstanta di Java:
   Gunakan keyword 'final' sebelum tipe data variabel.
   
   Sintaks:
   final tipeData namaKonstanta = nilai;
   
   Contoh:
   final double faktorAktivitas = 1.2;
   final int HARI_DALAM_MINGGU = 7;
   final String NAMA_APLIKASI = "HealthApp";
   
   Best Practice:
   - Untuk konstanta class-level (static), gunakan: public static final
   - Nama konstanta biasanya ditulis dalam HURUF_BESAR_DENGAN_UNDERSCORE
